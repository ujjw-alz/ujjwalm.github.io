My Projects

Project 1: E-commerce Website

This was a full-stack project I built for a client who wanted to sell products online. I used Node.js, Express, and MongoDB for the back-end and React for the front-end. The site was designed to be user-friendly and responsive, and it includes features such as shopping cart, payment processing, and order management.

Project 2: Social Media Platform

I worked on a team to create a social media platform that allows users to connect with friends, share content, and discover new interests. I contributed to the development of the front-end using Angular and designed the user interface. The platform also includes features such as real-time messaging, notifications, and a news feed.

Project 3: Mobile Game

I developed a mobile game for iOS and Android using Unity. The game is a casual puzzle game that challenges players to match colored blocks and reach the highest score. I was responsible for programming the game logic and implementing the user interface using C#.
